# Russian Language Sentiment Analysis

**Tech Stack:** Python (version 3.7+), Pandas, NumPy, NLTK ('punkt', stopwords, SnowballStemmer), TfidfVectorizer, scikit-learn (MultinomialNB)  
**GitHub URL:** [Project Link](#)

---

## Overview
This project focuses on performing **sentiment analysis on Russian-language text data**. The goal is to classify text into sentiment categories using classical NLP techniques and machine learning.

The workflow includes:  
- **Text preprocessing**: Tokenization using `punkt`, stopword removal, and stemming using `SnowballStemmer`.  
- **Feature extraction**: Converting text to numerical vectors using **TfidfVectorizer**.  
- **Model training**: Building a **Multinomial Naive Bayes (MultinomialNB)** classifier for sentiment prediction.

---

## Key Features
- Effective preprocessing pipeline for Russian-language text.  
- Sentiment classification using classical NLP techniques.  
- Demonstrates application of **Naive Bayes** for text classification tasks.
